# HTML
* textarea 屬性
直 row 橫 cols 
```css
  textarea {
    padding: 8px 16px;
    resize: vertical; /* none , horizontal*/
    max-height: 300px;  /* 可限制水平垂直拉動的大小 */
    min-height: 200px;
  }
  textarea {  }
```
* input 預設 date / time
用 [type="time"] 去選
date格式 : 2018-06-04 (YYYY-MM-DD)
time格式 : 15:30  (hh:mm)
