# 自動產生 Line Height
* 有寫 Line Height 
Height 可不用寫
* 用此好處 行高級距固定
```scss
  $baseLine: 10px;
  $lineHeightRatio: 1.2;

  @function rhythm($level) {
    @return $baseLine * ceil($level);
  }

  @function line($size) {
    @return rhythm($size * $lineHeightRatio / $baseLine);
  }

  @mixin font($size, $line-height: auto) {
    $line: line($size);

    font-size: $size;
    @if $line-height == auto or $line-height < $line {
      line-height: $line;
    } @else {
      line-height: $line-height;
    }
  }
```
