# 字型相關
* 避免載入過多種字型，導致載入速度降低

## CSS Font 相關屬性
```css
h1 {
  /* 可多各自行串接 前一個無效自動套用下一個 */
  font-family: Lobster, monospace;
}
```

## serif(襯線體) sans-serif(無襯線)
> https://blog.justfont.com/2014/06/google-fonts-2/
襯線 一種字型裝飾 (每個字母的微小突出)
無襯線 (類麥克筆、海報較為現代)


## GoogleFont
> https://fonts.google.com/

- 點擊+號
- 選擇粗細
- EMBED 將 <link~> 放入HTML
- 添加 CSS 屬性 
font-family: 'Lobster',cursive;
