# 客制化cmd

# 安裝 HyperJS
下載 https://hyper.is/

# HyperJS 
> https://hyper.is/plugins?featured
還有一堆插件可以額外安裝

# theme 可在此選擇
https://hyperthemes.matthi.coffee/

1. 把下面的設定加入 ~/.hyper.js
> 打開 HyperJS / Preference 即可
plugins: ['hyper-pokemon']

2. 輸入 hyper i hyper-pokemon

# hyper-pokemon 設定

    poketab: 'true', 
  }
