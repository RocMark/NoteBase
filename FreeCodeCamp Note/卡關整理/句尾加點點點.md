# 句尾依限制字數改成....

//! 如何判斷是否為單字

# 初版
```js
    function truncateString(str, num) {
        if (num >= str.length) {
            return str
        } 
        let result = str.split('').slice(0, num).join('')
        return `${result}...`
    }
    console.log(truncateString('A-tisket a-tasket A green and yellow basket', 11))
```
