# windows 指令
在輸入檔名時，不一定要打完
可按下 tab鍵 他會幫忙補齊
* help
* 在句尾加 /s

* 建立資料夾 md testDir
* 刪除資料夾 rd testDir

## 增刪改檔案
* 建空白檔案 
會產生有一行空白的
echo.> test.txt
* type nul>file.txt
可產生完全無檔案大小的檔案
* 刪除檔案 del test.text
del *.txt
* type 可查看內容
* move [file] [dir] 移動檔案 
* copy [file1] [file2] 複製/複寫檔案內容
* rename [file1] [file2] 重新命名
