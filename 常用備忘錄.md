# splice用法
//? 先取得該項目的 index
//* 執行 [].splice(i,1)
//* 即可刪除該項目資料，記得重新render頁面
```js
    let arr = [1, 2, 3, 4]
    arr.splice(1, 2) 
    // 從 1開始 刪除該index[1] & 後一個 共兩個
    //? 1為index[1] 2為數量
    console.log(arr) //* [ 1, 4 ]
```
