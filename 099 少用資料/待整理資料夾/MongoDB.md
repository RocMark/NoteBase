# Mongo 介紹
* MongoDB NoSQL數據庫[運行速度快、資料擴充易(免設scheme)]
* Shell操作 (JS客戶端/引擎)

# BSON
* BSON 二進制形式的存儲格式，具輕量、高效，可以有效描述非結構化數據
* JSON格式的擴展，JSON對於浮點、日期、特殊類型較無法滿足需求
* BSON 增加JSON數據類型，並轉換成二進制碼
