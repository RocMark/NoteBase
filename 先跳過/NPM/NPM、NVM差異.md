# NPM、NVM差異

## Node Version Manager
Node.js 版本管理的工具，切換各版本Node.js
- nvm install/uninstall [version]
- nvm ls 列表

### NVM 安裝檔
https://github.com/coreybutler/nvm-windows/releases

## Node Package Manager
用於安裝專案相依插件
- npm init 
初始化產生 package.json
- npm install 
安裝 package.json 設置的插件
- npm update
更新所有 Local Package
