# User Story
> 將 project拆成多個小步驟
Ex: 
```js
    //* Step1 User Story
    // 代辦事項app 需要哪些功能??
    // 1. 清單存放代辦事項
    // 2. 顯示給使用者
    // 3. 增加代辦事項
    // 4. 更改代辦事項
    // 5. 刪除代辦事項

    //* Step2  撰寫 pseudo code
    let todoList = []        // 1.
    console.log(todoList)    // 2.
    todoList.push('newItem') // 3.
    todoList = 'editItem'    // 4. 
    todoList.splice(0,1)     // 5.

    //* Step3 優化、整理、撰寫

    let todoList = {
        todos: ['item1', 'item2'],
        displayTodos() {
            console.log(this.todos)
        },
        addTodos(todo) {
            this.todos.push(todo)
            this.displayTodos()
        },
        changeTodo(position, todoText) {
            this.todos[position] = todoText
            this.displayTodos()
        },
        deleteTodo(position) {
            this.todos.splice(position, 1)
            this.displayTodos()
        },
    }
```
