# 句尾依限制字數改成....

... 也要算入字數

```js
    function truncateStr(str, num) {
        var count = (num<=3) ? num : num-3;
        
        var result = str.split('').slice(0, count).join('');
        if (num >= str.length) {
            return str;
        } else {
            return result + '...';
        }
    }
    truncateStr('A-', 1);
```
